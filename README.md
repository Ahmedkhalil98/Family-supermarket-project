## 📚 description
An application that includes online supermarket shopping,
ordering of meat, vegetables, fruits, etc., and having it delivered to your home.

### 👨‍💻 Languages and framework used:
 Dart, Flutter, PHP, MYSQL, Firebase
- [X] Firebase to register a new user and login with email or google accounts, and to register a new delivery driver and User.
- [X] MYSQL to store data for categories, products, orders and sliders .
